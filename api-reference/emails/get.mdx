---
title: 'Send Email'
openapi: 'POST /api/send-email'
---

Use this endpoint to send a templated email with dynamic data mapping.

## Request Body

The request must include the following fields:

| Field         | Type     | Required | Description                                             |
|---------------|----------|----------|---------------------------------------------------------|
| `api_key`     | string   | ✅       | Your iSend.ai API key for authentication               |
| `to`          | string   | ✅       | Recipient's email address                              |
| `from`        | string   | ✅       | Sender's email address                                 |
| `subject`     | string   | ✅       | Subject line of the email                              |
| `template`    | string   | ✅       | Template name to use                                   |
| `data_mapping`| object   | ✅       | Key-value pairs used to populate the email template     |

### Example Request

```bash
curl -X POST https://www.isend.ai/api/send-email \
  -H "Content-Type: application/json" \
  -d '{
    "api_key": "YOUR_API_KEY",
    "to": "delivered@resend.dev",
    "from": "bramu.ss@gmail.com",
    "subject": "hello world",
    "template": "welcome_emailer",
    "data_mapping": {
      "name": "John Doe",
      "email": "john@example.com"
    }
  }'
